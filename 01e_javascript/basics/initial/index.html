Übungs-Umgebung

Diese HTML-Seite dient dazu, fokussiert den Umgang mit Javascript als Programmiersprache üben zu können. Es sind keine Änderungen an der HTML-Datei notwendig. Sie können sich voll und ganz auf die Entwicklung des Javascript-Codes in der Datei js/spielwiese.js konzentrieren.

Die Übungs-Umgebung ist so konstruiert, dass der Einstiegspunkt für den Javascript-Teil immer die Funktion run() in der Datei js/spielwiese.js ist. Diese Funktion wird automatisch in folgenden Fällen aufgerufen:

  • Die HTML-Seite wird neu geladen.
  • Der unten stehende Button wird geklickt.

Bitte beachten Sie, dass Änderungen am Javascript-Code erst nach einem neu laden der HTML-Seite wirksam werden. Sollten Sie lediglich die Funktion ohne Änderungen am Javascript-Code erneut ausführen wollen, so bietet sich die Nutzung des Buttons an. Am Ende der Seite finden Sie zusätzlich eine Historie der Ausführungen seit dem letzten neu laden der HTML-Seite.

Ausgaben aus dem Javascript-Code erfolgen auf der Entwickler-Konsole. Öffnen Sie dazu bitte die Entwicklerwerkzeuge Ihres Browsers und navigieren Sie dort zur Konsole.
