Die nachfolgenden Aufgaben dienen dem Erlernen von CSS.

Aufbau von CSS-Anweisungen

Mit Hilfe verschiedener Selektoren können Bestandteile des HTML-Dokuments ausgewählt werden, um sie im nächsten Schritt zu gestalten.

.wichtig {
   text-decoration: underline;
}

Selektoren und Gestaltung im Allgemeinen

  1. Die Selektion kann mittels .meine-klasse anhand der Klasse namens meine-klasse erfolgen. Diese Art der Selektion ist gut geeignet, wenn mehrere Elemente auf einer Seite ähnlich dargestellt werden sollen. Testen Sie das bitte, indem Sie den Elementen mit der Klasse selektor die Farbe blue und Fettschrift zuordnen. Den Elementen mit der Klasse gestaltung weißen Sie bitte die Farbe darkred und Fettschrift zu.
  2. Die Selektion kann anhand einer ID erfolgen. Im Falle eines Elements mit der ID meine-id erfolgt die Selektion über #meine-id. Testen Sie das bitte, indem Sie dem Element mit der ID test-id die Farbe orange und Fettschrift zuweisen.
  3. Möchte man alle Elemente selektieren, die einen bestimmten Tag wie z.B. <h2> verwenden, so lautet der Selektor im Beispiel h2. Testen Sie das bitte, indem Sie den Text aller h2-Elemente unterstreichen.
  4. Mit Hilfe von Pseudoklassen können Zustände von Elementen bei der Selektion berücksichtigt werden. Häufig wird das z.B. für Links verwendet. Der Zustand eines Link über dem der Mauszeiger steht, wird über a:hover selektiert. Testen Sie das bitte, indem Sie alle Links in der Farbe red und unterstrichen darstellen, sobald sich der Mauszeiger über dem Link befindet.
  5. Es können auch mehrere Selektoren für die gleichen Anweisungen verwendet werden, indem die Selektoren mit "," aneinandergereiht werden. Diese werden logisch gesehen mit "oder" verknüpft. Ein Beispiel ist .eine-klasse, .andere-klasse. Testen Sie das bitte, indem Sie allen Elementen mit der Klasse erste-klasse oder zweite-klasse die Farbe green und Fettschrift zuweisen. Kombinieren Sie dazu die beiden Selektoren entsprechend.
  6. Ein Selektor kann auch aus mehreren Bestandteilen zusammengesetzt sein, die eine "und" Beziehung abbilden. Dazu werden die beiden Selektoren ohne Leerzeichen aneinandergereiht. Ein Beispiel ist .erste-klasse.zweite-klasse. In diesem Fall werden nur Elemente selektiert, denen beide Klassen gleichzeitig zugeordnet sind
    7. . Testen Sie das bitte, indem Sie allen Elementen mit den gleichzeitig zugewiesenen Klassen erste-klasse und zweite-klasse die Farbe purple und Fettschrift zuweisen.

CSS Box-Model

Die folgenden Arbeitsschritte beziehen sich alle auf das Element mit der ID box.

Das ist der Beispieltext.

  1. Erstellen Sie einen Rahmen mit einer Breite von 10px, bei dem die Linie durchgezogen ist und der die Farbe red hat.
  2. Der Text soll einen Abstand von mindestens 30px vom Rahmen haben.
  3. Der Rahmen soll einen Abstand von 30px von allen anderen Elementen haben.
  4. Zentrieren Sie den Text horizontal in der Box.

Layout

  1. Lassen Sie das div Element mit der ID
    inline-div
    inline darstellen, anstatt wie gewöhnlich als Blockelement.
  2. Richten Sie das Element mit der ID flexbox mit Hilfe des Flexbox Layouts so aus, dass die Elemente nebeneinander anstatt untereinander angezeigt werden.
  3. Richten Sie das Element mit der ID grid mit Hilfe des Grid Layouts so aus, dass die Elemente in zwei Spalte mit je 200px Breite dargestellt werden.
  4. Positionieren Sie den nachfolgenden Link zum Anfang des Dokuments so, dass der Link zu jedem Zeitpunkt 100px vom unteren und 100px vom rechten Rand positioniert ist.Start
