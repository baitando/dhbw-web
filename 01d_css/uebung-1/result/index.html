Übungen zum Thema CSS

Die nachfolgenden Aufgaben dienen dem Erlernen von CSS.

Selektoren und Gestaltung im Allgemeinen

  1. Selektieren Sie alle Elemente, denen die Klasse "meine-klasse" zugeordnet ist. Weißen Sie diesen Elementen blaue Schriftfarbe und Fettschrift zu.

    Test: Dieser Text sollte blaue Schriftfarbe und Fettschrift verwenden.

  2. Selektieren Sie alle Elemente mit der ID "meine-id". Weißen Sie diesen Elementen orange Schriftfarbe und Fettschrift zu.

    Test: Dieser Text sollte orange Schriftfarbe und Fettschrift verwenden.

  3. Selektieren Sie alle Elemente, die den Tag "h2" verwenden. Weißen Sie diesen Elementen eine unterstrichene Schrift zu.

    Test: Betrachten Sie unter Anderem die Überschrift dieses Aufgabenblocks und prüfen Sie, ob diese unterstrichen ist.

  4. Selektieren Sie alle Hyperlinks im Zustand, in dem sich der Mauszeiger auf dem Hyperlink befindet. Weißen Sie diesen Elementen rote Schriftfarbe zu.

    Test: Link

  5. Selektieren Sie alle Elemente, denen die Klasse "erste-klasse" oder "zweite-klasse" zugewiesen sind. Weißen Sie diesen Elementen grüne Schrift farbe und Fettschrift zu.

    Test: Sowohl diese Hälfte des Textes als auch diese Hälfte des Textes sollte grüne Schriftfarbe und Fettschrift verwenden.

  6. Selektieren Sie alle Elemente, denen die gleichzeitig die Klassen "erste-klasse" und "zweite-klasse" zugewiesen sind. Weißen Sie diesen Elementen lila Schriftfarbe und Fettschrift zu.

    Test: Dieser Text sollte lila Schriftfarbe und Fettschrift verwenden.

CSS Box-Model

Die folgenden Arbeitsschritte beziehen sich alle auf das Element mit der ID "box".

Das ist der Beispieltext.

  1. Erstellen Sie einen Rahmen mit einer Breite von 10px, bei dem die Linie durchgezogen ist und der die Farbe red hat.
  2. Stellen Sie sicher, dass der Text einen Abstand von mindestens 30px vom Rahmen hat.
  3. Stellen Sie sicher, dass die einen Abstand von mindestens 30px zu den angrenzenden Elementen hat.
  4. Zentrieren Sie den Text horizontal in der Box.

Layout

  1. Lassen Sie das div Element mit der ID
    inline-div
    inline darstellen, anstatt wie gewöhnlich als Blockelement.
  2. Richten Sie das Element mit der ID flexbox mit Hilfe des Flexbox Layouts so aus, dass die Elemente nebeneinander anstatt untereinander angezeigt werden.
  3. Richten Sie das Element mit der ID grid mit Hilfe des Grid Layouts so aus, dass die Elemente in zwei Spalte mit je 200px Breite dargestellt werden.
  4. Positionieren Sie den nachfolgenden Link zum Anfang des Dokuments so, dass der Link zu jedem Zeitpunkt 100px vom unteren und 100px vom rechten Rand positioniert ist.Start
